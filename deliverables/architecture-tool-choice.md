# Architecture Diagram - Which Path Is Easier?

## Option 1 - Excalidraw

Best when:
- you want a polished submission image
- you want visual control over layout and emphasis
- you are comfortable manually placing boxes/arrows

Use:
- `deliverables/architecture-excalidraw-layout.md`

Pros:
- better-looking final PNG
- easier to tune spacing, colors, and callouts
- best for final judging visuals

Cons:
- slightly slower to build manually

## Option 2 - Mermaid

Best when:
- you want a quick first visual reference
- you want to test the structure before drawing manually
- you may later trace/copy the structure into Excalidraw

Use:
- `deliverables/architecture.mmd`

Pros:
- faster to generate a draft
- easy to validate overall flow

Cons:
- less visually refined for final submission
- note/callout styling is limited

## Recommended Path

1. Open `deliverables/architecture.mmd` to validate the logic and flow.
2. Build the final submission version in Excalidraw using `deliverables/architecture-excalidraw-layout.md`.
3. Export final PNG to `docs/architecture.png` or `deliverables/architecture-final.png`.

## Time-Saving Tip

If you are short on time:
- skip decorative icons
- use simple rounded rectangles
- keep callouts minimal
- focus on readability over artistry
